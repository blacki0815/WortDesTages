# WortDesTages

**WortDesTages** is a macOS application that shows you a new word every day directly on your desktop.  
The word is selected from a category of your choice and is meant to expand your knowledge in a simple and unobtrusive way.

## About the App

This app displays one **daily word** from a selected category on your macOS desktop.  
It is designed to be lightweight, easy to use, and helpful for learning something new every day.

### Available Categories
- **Economy**
- **History**
- **General Knowledge**

Each day, the app suggests a new word from the chosen category.

## Built With
- **Swift**
- **Xcode**
- **macOS**
- Includes a **Widget Extension** for better desktop integration

## Installation

1. Clone or download this repository.
2. Open the project in **Xcode**.
3. Build and run the app on any macOS system.
4. Allow the necessary permissions if macOS asks for them.

> The app can be installed locally on any macOS machine via Xcode, as usual for macOS development projects.

## Testing

The project includes:
- Unit Tests
- UI Tests

These can be run directly from Xcode.

## Project Structure

- `WortDesTages` – Main application
- `WortDesTagesTests` – Unit tests
- `WortDesTagesUITests` – UI tests
- `WortDesTagesWidgetExtension` – Widget extension
